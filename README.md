## Set up
This is a Python 3 project.

1. Clone the repo

2. Create a virtual env and start it

```
python3 -m venv env
source env/bin/activate
```

3. Install the PIP requirements

```
pip install -r requirements.txt
```

## Run the script
Activate the virtualenv and run

```
python roller.py
```

Output goes in `output/`
